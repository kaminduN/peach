 11  Fuzzy C-Means 
 12   
 13  Fuzzy C-Means is a clustering algorithm based no fuzzy logic. 
 14   
 15  This package implements the fuzzy c-means algorithm for clustering and 
 16  classification. This algorithm is very simple, yet very efficient. From a 
 17  training set and an initial condition which gives the membership values of each 
 18  example in the training set to the clusters, it converges very fastly to crisper 
 19  sets. 
 20   
 21  The initial conditions, ie, the starting membership, must follow some rules. 
 22  Please, refer to any bibliography about the subject to see why. Those rules are: 
 23  no example might have membership 1 in every class, and the sum of the membership 
 24  of every component must be equal to 1. This means that the initial condition is 
 25  a fuzzy partition of the universe. 
 26  """
