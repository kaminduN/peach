 11  __doc__ = """ 
 12  This package implements operations of fuzzy logic. 
 13   
 14  Basic operations are ``and (&)``, ``or (|)`` and ``not (~)``. Those are 
 15  implemented as functions of, respectively, two, two and one values. The ``and`` 
 16  is the t-norm of the fuzzy logic, and it is a function that takes two values and 
 17  returns the result of the ``and`` operation. The ``or`` is a function that takes 
 18  two values and returns the result of the ``or`` operation. the ``not`` is a 
 19  function that takes one value and returns the result of the ``not`` operation. 
 20  To implement your own operations there is no need to subclass -- just create the 
 21  functions and use them where appropriate. 
 22   
 23  Also, implication and aglutination functions are defined here. Implication is 
 24  the result of the generalized modus ponens used in fuzzy inference systems. 
 25  Aglutination is the generalization from two different conclusions used in fuzzy 
 26  inference systems. Both are implemented as functions that take two values and 
 27  return the result of the operation. As above, to implement your own operations, 
 28  there is no need to subclass -- just create the functions and use them where 
 29  appropriate. 
 30   
 31  The functions here are provided as convenience. 
 32  """
