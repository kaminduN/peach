 10  __doc__ = """ 
 11  Learning rules for neural networks and base classes for custom learning. 
 12   
 13  This sub-package implements learning methods commonly used with neural networks. 
 14  There are a lot of different topologies and different learning methods for each 
 15  one. It is very difficult to find a consistent framework for defining learning 
 16  methods, in consequence. This method defines some base classes that are coupled 
 17  with the neural networks that they are supposed to work with. Also, based on 
 18  these classes, some of the traditional methods are implemented. 
 19   
 20  If you want to implement a different learning method, you must subclass the 
 21  correct base class. Consult the classes below. Also, pay attention to how the 
 22  implementation is expected to behave. Since learning algorithms are usually 
 23  somewhat complex, care should be taken to make everything work accordingly. 
 24  """
