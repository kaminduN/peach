 86          is not recomended that different investigations are used with the same 
 87          optimizer in the same cost function. 
 88   
 89          :Parameters: 
 90            x 
 91              The value from where the new estimate should be calculated. This can 
 92              of course be the result of a previous iteration of the algorithm. 
 93   
 94          :Returns: 
 95            This method returns a tuple ``(x, e)``, where ``x`` is the updated 
 96            estimate of the minimum, and ``e`` is the estimated error. 
 97          ''' 
 98          # Initializes the inverse hessian, if needed 
 99          if self.__B is None: 
100              B = inv(hessian(self.__f)(x)) 
101          else: 
102              B = self.__B 
103

105          n = x.size 
106          x = x.reshape((n, 1))          # x as a line-vector

109          xn = x + dx 
110

113          Bty = dot(B.T, y)
