Class Trapezoid

source code

Trapezoid function.

Given four points, x0, x1, x2 and x3, with x0 < x1 < x2 < x3, creates a function which returns:

0, if x <= x0 or x > x3;

(x - x0)/(x1 - x0), if x0 <= x < x1;

1, if x1 <= x < x2;

(x3 - x)/(x3 - x2), if x2 <= x < x3.
