213          ''' 
214          The feedforward method to the layer. 
215   
216          The ``__call__`` interface should be called if the answer of the neuron 
217          to a given input vector ``x`` is desired. *This method has collateral 
218          effects*, so beware. After the calling of this method, the ``v`` and 
219          ``y`` properties are set with the activation potential and the answer of 
220          the neurons, respectivelly. 
221   
222          :Parameters: 
223            x 
224              The input vector to the layer. 
225   
226          :Returns: 
227            The vector containing the answer of every neuron in the layer, in the 
228            respective order. 
229          ''' 
230          # Adjusts the input vector in case the neuron is biased. Also, the 
231          # input vector is reshaped as a column-vector. 
232          x = reshape(x, (self.inputs, 1)) 
233          if self.__bias:
